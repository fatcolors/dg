DataGEMS case study — screen placeholders (desktop app)
========================================================

Drop your exported Figma screens into THIS folder using the exact filenames
below. They replace the dashed placeholders in index.html automatically —
no code changes needed.

Each screen sits in a desktop "window" frame that BLEEDS off the right edge
of the slide, so the LEFT part of the screen is what stays visible. Export
wide desktop screens (16:10 works best). The image is anchored to its
top-left, so keep the most important content on the left side of the screen.
PNG, exported at 2x, is recommended.

Slide 2  · Context ............... platform.png        (the app / dashboard)
Slide 3  · The problem ........... problem.png         (data discovery view)
Slide 5  · Research / listening .. research-team.png   (notes, affinity map…)

Slide 12 · Hand-off / QA ......... qa.png               (build vs design)
Slide 13 · Outcome ............... outcome.png          (final UI)

Slides 1, 4, 14 and 15 (cover, the 70+ stat, reflection, closing) have no
screen by design. If any file is missing, that slide just shows a labelled
placeholder, so you can fill them in any order.
